# Tesla Crypt

Sistema de cifrado de archivos extremo a extremo (E2EE).

## Arquitectura

Modelo híbrido:

- AES-256-GCM para cifrado de archivos.
- RSA-3072-OAEP para protección de claves.

## Tecnologías

Python 3
Cryptography
PyCryptodome
